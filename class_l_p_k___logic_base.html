LPK_LogicBase.OnEvent ( LPK_EventManager.LPK_EventData  data)
protectedvirtual

Empty function overriden to handle script action implementation. Detaching this function from the event system will be done by default. Any other functions the user makes in their own components to hook up with the event system must be removed manually through the class's OnDestroyed function (not to be confused with OnDestroy).
